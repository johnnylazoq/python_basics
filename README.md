# Python Basics

A simple repository for practicing Python fundamentals.

## Goal

Use this project to practice core Python concepts, including:
- Variables and data types
- Conditionals
- Loops
- Functions
- Lists and dictionaries
- Basic file handling

## Project Setup

1. Move into the project folder:

   cd /home/johnny/lewagon/prep-work/python_basics

2. Activate the virtual environment:

   source .venv/bin/activate

3. Check Python version:

   python --version

## Suggested Workflow

1. Create exercise files such as:
   - exercises/01_variables.py
   - exercises/02_conditionals.py
   - exercises/03_loops.py
2. Run files from the terminal:

   python exercises/01_variables.py

3. Commit your progress regularly with clear commit messages.

## Environment Notes

- The virtual environment is stored in .venv.
- .venv is ignored by git via .gitignore.

If you later need credentials for any APIs, create a local .env file and add .env to .gitignore.
